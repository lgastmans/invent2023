Batches

The batches window lets you get an overview of all batches for the currently selected month in the current storeroom.
The window contains a navigation bar at the top, and below it a grid with all relevant batch details. You can change the sorting of the grid by clicking on the column header.
The following buttons elements are in the navigation:

ElementDescription
Brings up the Transfer Stock window that lets you move stock for the selected batch between storerooms or out of the stock management system.
Shows or hides deleted batches. For Administrators Only
Brings up the print view for the current data
FilterLets you filter the details in the grid. The filter box has three fields:
Filter Field - The column to filter
Filter Type - can be starts with, ends with, contains, or equals.
Filter Value - value to filter. Make this blank to reset filter.
Filter button - applies the filter
NavigationLets you navigate through the results
Go the the first page
Go the the previous page
Go the the next page
Go the the last page
RowsLets you change the number of rows displayed per page
ViewsLets you change the current view of the grid. The default view is alwas shows but can be customized using the change view button.
Brings up the change view window to edit the current view
Refreshes the contents of the grid
